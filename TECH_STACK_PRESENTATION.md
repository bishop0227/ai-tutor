# AI Tutor Web Application - Technology Stack

## Frontend Technologies

• **React** - Component-based UI framework for building interactive user interfaces
• **Vite** - Fast build tool and development server for rapid development experience
• **TypeScript** - Type-safe JavaScript for catching errors early and improving code maintainability
• **Tailwind CSS** - Utility-first CSS framework for rapid UI styling and consistent design
• **Shadcn UI** - High-quality, accessible component library built on Radix UI primitives
• **React Router** - Client-side routing for seamless navigation in single-page application
• **Axios** - HTTP client for making API requests to the backend server

## Backend Technologies

• **Python Flask** - Lightweight web framework ideal for AI integration and rapid API development
• **Flask-CORS** - Cross-origin resource sharing support for frontend-backend communication
• **Gunicorn** - Production-ready WSGI server for deploying Flask applications
• **SQLAlchemy** - ORM for database operations with SQLite, enabling easy data management

## AI & Machine Learning

• **Google Gemini API** - Multimodal LLM for PDF summarization, quiz generation, and study planning
• **LangChain** - Framework for orchestrating AI workflows and managing context between PDF content and LLM
• **FAISS** - Vector similarity search library for efficient retrieval of relevant PDF content
• **Sentence Transformers** - Embedding models for converting text into vector representations
• **PyPDF2 / pypdf** - PDF parsing libraries for extracting text content from uploaded documents

## Database

• **SQLite** - Lightweight relational database for storing user profiles, quiz history, and learning analytics

## Deployment

• **Vercel** - Frontend hosting platform with automatic deployments and global CDN
• **Render** - Backend hosting platform for Python Flask applications with easy scaling
